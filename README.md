# ZSH Config

## Scripts used:
If you want to download the scripts I've used in this config, [click here](https://github.com/danhat1020/scripts).

**Steps to replicate:**
1. Move ~/.zshrc file to .config/zsh/zshrc
2. Create new ~/.zshrc
3. Write:
```
if [ -r ~/.config/zsh/zshrc ]; then
  source ~/.config/zsh/zshrc
fi
```
